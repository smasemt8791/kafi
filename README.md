### النسخة العربية الكاملة (README.md)

كافي – Kafi PWA  
المُحلل المالي الذكي وصانع دراسات الجدوى الاحترافية في جيبك

#### نبذة عن المشروع
«كافي» هو تطبيق ويب تقدمي (PWA) متكامل وحديث 100% لإعداد دراسات الجدوى المالية والاقتصادية بطريقة ذكية وبسيطة amazingly سريعة.

سواء كنت رائد أعمال، مستثمر، صاحب مشروع صغير أو متوسط، أو حتى طالب جامعي يُعد مشروع تخرج، كافي يساعدك على تحويل فكرتك إلى دراسة جدوى احترافية كاملة في دقائق معدودة.

تم تطويره بواسطة أخصائي معلوماتية صحية ومالية ومطور Full-Stack، مع التركيز على الدقة العالية، الخصوصية التامة، والعمل بدون إنترنت.

#### الميزات الرئيسية
- إنشاء دراسات جدوى مالية كاملة من الصفر في أقل من 10 دقائق  
- رسوم بيانية احترافية
- تقرير نهائي جاهز للطباعة أو التصدير PDF بتصميم أنيق  
- يعمل 100% بدون إنترنت بعد التثبيت الأول  
- قابل للتثبيت على الهاتف والكمبيوتر كتطبيق أصلي  
- وضع ليلي + تصميم متجاوب فائق السرعة

#### رابط التطبيق المباشر
https://smasemt8791.github.io/kafi/

#### التقنيات المستخدمة
- React + TypeScript  
- Vite (أسرع أداة بناء حاليًا)  
- Tailwind CSS  
- Recharts & Chart.js  
- Zustand لإدارة الحالة  
- LocalForage + IndexedDB  
- jsPDF + html2canvas لتصدير PDF  
- Full PWA (Service Worker + Manifest + Push-ready)

#### تعليمات التشغيل للمطورين
```bash
git clone https://github.com/smasemt8791/kafi.git
cd kafi
npm install
npm run dev        # تشغيل التطوير
npm run build      # بناء النسخة النهائية
```

حوّل فكرتك إلى مشروع ناجح بدراسة جدوى احترافية ودقيقة  
كافي معك في كل خطوة

### English Version (Full README)

**Kafi – Smart Financial Feasibility Analyzer PWA**  
The ultimate all-in-one tool to create professional feasibility studies in minutes

#### Project Overview
Kafi is a powerful, beautiful, and completely offline Progressive Web App that turns your business idea into a full-fledged, investor-ready financial feasibility study — fast, accurate, and private.

Whether you're an entrepreneur, investor, startup founder, SME owner, or student, Kafi gives you everything you need to validate your project financially before spending a single dollar.

#### Key Features
- Create complete financial feasibility studies in under 10 minutes  
- Professional interactive charts and graphs  
- Export full report as beautifully designed PDF (ready to present or print)  
- Works 100% offline after first visit  
- Installable as a native app on phone & desktop  
- Dark mode + blazing-fast performance
